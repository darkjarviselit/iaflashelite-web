# Adaptación sectorial — Agencia web

## Objetivo

Ordenar solicitudes de clientes que quieren una landing, web corporativa,
rediseño o mejora de presencia online.

## Qué pregunta el formulario

- Tipo de web que necesita.
- Objetivo principal: vender, captar leads, presentar marca, validar oferta.
- Páginas necesarias.
- Referencias visuales.
- Si tiene textos, logo, dominio y hosting.
- Plazo deseado.
- Presupuesto aproximado si lo sabe.

## Qué analiza la IA

- Si el cliente necesita landing, web corporativa o ecommerce.
- Si hay suficiente material para empezar.
- Qué falta: textos, identidad visual, estructura, dominio, hosting.
- Riesgos de alcance.
- Si conviene fase P0 antes de una web grande.

## Propuesta inicial posible

```text
Por lo que cuentas, parece que necesitas una landing clara para validar la
oferta antes de construir una web más grande.

Antes de cerrar alcance, necesito confirmar:

1. ¿Tienes textos y logo preparados?
2. ¿La web debe captar leads o solo presentar información?
3. ¿Tienes dominio y hosting?

El enfoque inicial podría ser una landing de una página con propuesta de valor,
bloques principales, CTA y formulario de contacto. Lo revisaría después de ver
referencias y materiales disponibles.
```

## Qué no prometer

- No prometer posiciones SEO.
- No prometer ventas.
- No prometer plazos si faltan textos o marca.
- No aceptar ecommerce complejo como si fuera landing simple.
