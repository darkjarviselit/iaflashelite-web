# Checklist final — Sistema IA Pro

Usa esta checklist al terminar el pack. No hace falta que todo sea perfecto. Sí
debe quedar claro que tienes una estructura de trabajo real.

## Sistema base

- [ ] Tengo mis 3 proyectos IA creados o definidos.
- [ ] Tengo contexto mínimo cargado en cada proyecto.
- [ ] Sé para qué sirve cada proyecto y cuándo usarlo.
- [ ] Tengo claro que un proyecto IA no es memoria perfecta, sino un espacio con
      contexto, instrucciones y archivos más estables que un chat suelto.

## Proyectos IA

- [ ] Proyecto 1: Estrategia y mejoras.
- [ ] Proyecto 2: Marketing y ventas.
- [ ] Proyecto 3: Prompt Builder técnico.
- [ ] Cada proyecto tiene instrucciones claras.
- [ ] Cada proyecto sabe qué no debe prometer ni decidir por mí.

## Flujo técnico

- [ ] Tengo mi Prompt 0.
- [ ] Sé usar el flujo candado.
- [ ] Hice una prueba real con una tarea pequeña.
- [ ] Sé pegar una respuesta de Claude Code/Codex de vuelta en mi proyecto IA.
- [ ] Sé revisar qué se tocó, qué se validó y qué riesgo queda.

## Cierre profesional

- [ ] Tengo una checklist de revisión de respuestas.
- [ ] Tengo una plantilla de informe final.
- [ ] Sé separar hechos, hipótesis, riesgos y próximos pasos.
- [ ] Sé cuándo pedir otra fase y cuándo parar.

## Siguiente paso

- [ ] Sé qué bot, agente o automatización pequeña podría construir después.
- [ ] Tengo claro que el Pack 3 será para crear un primer bot/agente funcional,
      no para prometer automatización sin revisión.
