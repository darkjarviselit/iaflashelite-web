# Audio guía — Sistema IA Pro

Guion fuente para futuro audio guía breve de acompañamiento.

Duración orientativa: 8-15 minutos.

No generar audio en esta fase.

## Apertura

Bienvenido a Sistema IA Pro, el segundo pack de IAFlashElite Academy.

En Pack Arranque IA dejamos preparada la base: quién eres, qué proyecto quieres
construir y cómo debe ayudarte tu IA. En Sistema IA Pro damos el siguiente paso:
crear proyectos IA que trabajen contigo como centro de mando.

La idea no es tener más chats. La idea es tener espacios de trabajo con contexto,
instrucciones y función clara.

## Qué vas a montar

Vas a montar tres proyectos IA:

El primero es Estrategia y mejoras. Sirve para pensar, priorizar, detectar
riesgos y decidir qué hacer primero.

El segundo es Marketing y ventas. Sirve para trabajar ofertas, mensajes, copy,
precios, contenido y posicionamiento.

El tercero es Prompt Builder técnico. Sirve para preparar prompts claros para
Claude Code, Codex u otras herramientas técnicas.

No necesitas crear los tres el mismo día si tu proyecto todavía es pequeño. Pero
sí conviene entender por qué existen.

## Si vienes del Pack Arranque IA

Si hiciste Pack Arranque IA, ya tienes tres piezas muy valiosas: tu Currículum
IA personal, tu Contexto Maestro del Proyecto y tus Instrucciones de Trabajo.

No las repitas desde cero. Úsalas como base para cargar contexto en tus
proyectos.

## Si empiezas directamente

Si no hiciste Pack Arranque IA, usa la plantilla de contexto mínimo rápido. No
será tan profunda como el pack anterior, pero te dará suficiente base para no
empezar desde una conversación vacía.

## Prompt 0

Prompt 0 es el arranque de cualquier trabajo técnico.

Antes de pedir a Claude Code o Codex que toque archivos, necesitas definir el
objetivo, el contexto, los límites, los archivos prohibidos, las validaciones y
el formato de reporte.

Prompt 0 no es decoración. Es una forma de proteger el trabajo.

## Flujo candado

El flujo candado es simple:

Primero, tu proyecto IA audita el problema y prepara el prompt.

Segundo, tú pegas ese prompt en Claude Code, Codex o la herramienta técnica.

Tercero, la herramienta ejecuta o propone cambios.

Cuarto, tú pegas la respuesta de vuelta al proyecto IA.

Quinto, el proyecto IA revisa qué pasó.

Sexto, tú decides el siguiente paso.

La frase clave es esta: tú eres el director de orquesta. La IA trabaja, pero tú
lees, decides y mantienes el control.

## Revisión

No cierres un trabajo solo porque la herramienta diga que está terminado.

Revisa qué hizo, qué tocó, qué no tocó, qué validó, qué riesgo queda y si falta
otra fase.

Esta parte parece lenta, pero ahorra errores.

## Cierre

Cuando termines, usa la plantilla de informe final. No hace falta escribir una
novela. Basta con objetivo, cambios, evidencias, validaciones, riesgos,
próximos pasos y lo que no se tocó.

Eso convierte una sesión de IA en un trabajo entendible.

## Cierre del audio

Sistema IA Pro no promete que la IA lo haga todo. Te enseña a dirigirla mejor.

Cuando ya tengas tus proyectos, tu Prompt 0 y tu flujo candado, estarás listo
para el siguiente nivel: construir algo funcional.

Eso será **Primer Sistema IA Vendible**.
